Siguiente : Roles de usuario


El navegador busca la url el nombre del controller, metodo, parametros
Ej:
http://localhost/betel04/login/
Busca el archivo  Controllers/Login.php, 
como no se esta especificando el metodo, busca uno con el mismo nombre
en este caso login


Crear un nuevo controlador:
1.- Carpeta Controllers
2.- Archivo iniciando en mayuscula ej: Clientes
3.- metodo con mismo nombre en minusculas ej: clientes 

Crear vista
1.- Carpeta Views
2.- Carpeta con nombre en minuscula ej: clientes
3.- Crear archivo con nombre en minuscula ej: clientes
